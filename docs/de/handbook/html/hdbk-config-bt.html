Schriften

BibleTime benutzt die Standard-Schriftart von KDE. Diese kann im KDE Control Center eingestellt werden. Wenn ein Modul mit der Standardschriftart nicht richtig dargestellt wird, wählen Sie bitte eine andere Schriftart im Konfigurationsdialog. BibleTime ordnet Sprachen eigene Schrifteinstellungen zu.

Der Optionendialog - Schriften.

Welche Typen von Schriften können benutzt werden?

BibleTime kann nun jede installierte Schriftart benutzen. Diese können Bitmap-Schriftarten, Adobe® Type1 Schriftarten, Adobe® Type3 Schriftarten, oder TrueType® Schriftarten sein.Die Schriftarten sind entdweder mit ihrer Distribution geliefert worden, oder können seperat bekommen werden.

Solange Ihre benötigten Module korrekt angezeigt werden, muss hier nichts getan werden. Wenn ein Modul als eine Folge von Fragezeichen (???????????????????) oder als leere Kästchen angezeigt wird, dann entht die für BibleTime verfügbare Schriftart nicht die im Dokument benutzten Zeichen.

Wählen Sie im Schriftarten-Abschnit des Konfigurationsdialoges die gewünschte Sprache im Aufklappmenü. Klicken Sie auf die "Eigene Schriftart benutzen" Box. Wählen Sie nun eine Schriftart. Clearlyu unterstüzt viele Sprachen, diese könnte ein erster Versuch sein. Wenn keine Schriftart gefunden werden konnte, die das gewünschte Modul korrekt anzeigt, versuchen Sie bitte das KDE Sprachpaket für die gewünschte Sprache zu installieren.

Schriftarten installieren

Detallierte Anweisungen gehen über den Umfang dieses Handbuces hinaus. Wenn Sie genaue Anweisungen zur Installation von Schriftarten benötigen, lesen Sie bitte dasFont-HOWTO, und das Unicode HOWTO

Tip

Wenn ihre Distribution eine Werkzeug zum installieren von Schriftarten zur Verfügung stellt, benuzten Sie ihn! Die Installation wird so viel leichter.

Tip

Wenn Sie eine kleine Unicode-Schriftart wie clearlyu (etwa 22kb) benutzen, wird BibleTime schneller laufen, als wenn Sie eine umfangreiche Schriftart wie Bitstream Cyberbit (ungefähr 12Mb) verwenden.

Schriftarten bekommen

Schriftarten können Sie von unterschiedlichen Quellen kommen.

  • Ihre Distribution

  • Das jeweilige Sprachpaket ihrer Distribution.

  • Eine bestehende Microsoft® Windows® Installation auf dem gleichen Computer.

  • Eine Sammlung von Schriftarten, die von Adobe® oder Bitstream® zu bekommen sind.

  • Schriftarten-Sammlungen aus dem Internet.

Unicode Schriftarten benuzten mehr Zeichen als andere Schriftarten, sie sind also eine gute Wahl für BibleTime. Einige dieser Schriftarten sind kostenlos erhtlich. Nur wenige Schriftarten enthalten alle Zeichen, die im Unicode Standard definiert sind, so dass Sie evtl. mehrere ausprobieren müssen, bevor Sie das gewünschte Resultat erhalten.

Sitzungen

Profile definieren die Erscheinung des Arbeitsbereiches, z.B. welche Fenster geöffnet sind, und welche Texte in diesen Fenstern angezeigt werden sollen. Wenn eine Sitzung einmal erstellt ist, erwenden Sie den Eintrag Fenster -> Sitzung sichern, um die aktuelle Sitzung in eine Sitzungsdatei zu sichern. Verwenden Sie Fenster -> Sitzung laden, um eine bereits gespeicherte Sitzung in BibleTime zu laden.

Sword

Viele Funktionen, die von Sword bereitgestellt werden, können jetzt in BibleTime angepasst werden.

Der Konfigurationsdialog - Sword.

Sword Allgemein

Schlüsselcache für Lexika

BibleTime kann einen Schlüsselcache für Lexikons anlegen. Dies beschleunigt das Öffnen großer Lexika. Es verbraucht allerdings etwas Festplattenkapazität, also sollten Sie dieses Feature abschalten, wenn Sie nur über sehr wenig freien Speicher haben.

Scrollverhalten

Per Voreinstellung bewegt der Pfeil nach unten das Anzeigefenster zum nächsten Vers. Mit dieser Option kann man das Verhalten so ändern, dass es zum vorigen Vers springt.

Wählen Sie eine Sprache für die Bücher der Bibel

Diese Funktion erlaubt es ihnen, die Sprache zu ändern, in der die Buchnamen angezeigt werden. So kann die King James Bibel zum Beispiel mit deutschen Buchnamen angezeigt werden. Wenn ihre eigene Sprache nicht angezeigt wird, und Sie gerne eine Sword-Liste für ihre Sprache erstellen würden, dann schauen Sie bitte auf den Entwicklerseiten der Crosswire Bibelgesellschaft nach.

Sword Filtereinstellungen

Filter kontrollieren die Erscheinungsform des Textes in den Anzeigefenstern. Hier können Sie die Voreinstellungen für verschiedene Filter festlegen.

Folgendes kann durch Filter kontrolliert werden:

  • Zeilenumbruch nach jedem Vers anzeigen

  • Versnummern anzeigen

  • Fußnoten anzeigen

  • Strongnummern anzeigen

  • Überschriften anzeigen

  • Morphologische Tags anzeigen

  • Wurzeln anzeigen

  • Hebräische Vokalzeichen anzeigen

  • Hebräische Kantillationszeichen anzeigen

  • Griechische Akzente anzeigen
